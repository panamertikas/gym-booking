# GymBooking - Console Application

A gym booking management system built with Java OOP.

## Projects
- **Project 1** (this repo): Console-based application
- **Project 2**: JDBC/JPA/Hibernate - [gym-booking-jdbc](https://github.com/panamertikas/gym-booking-jdbc)
- **Project 3**: Spring Boot REST API (coming soon)

## Technologies
- Java 21
- OOP principles
- Design Patterns (Singleton, Builder, Factory Method)
- Layered Architecture (Model, DAO, Service)
